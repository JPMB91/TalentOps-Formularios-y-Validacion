# Formularios y Validación Avanzados

Sistema completo de formulario implementado con React y Node.js con soporte para subir imágenes comprimidas.

## Tecnologías

- React
- Express(para la subida de imágenes)
- Multer
- Axios
- Zod
- Json-server(para simular backend de usuarios)

## Características

- Validación de campos en tiempo real
- Display de errores inline
- Estados visuales para cada campo
- Optimistic UI para mejor UX
- Subida de imágenes con preview y barra de progreso del upload
- Compresión automática de imágenes
- Validación al hacer envío del formulario con Zod

## Prerequisitos

- Node.js (v14 o superior)
- npm o yarn

## Instalación

1. Clonar el repositorio:
```bash
git clone https://github.com/JPMB91/TalentOps-Formularios-y-Validacion
cd formularios-avanzados
```

2. Instalar dependencias del proyecto principal:
```bash
npm install
```

## Uso

### Iniciar todos los servidores simultáneamente

Desde la raíz del proyecto:
```bash
npm run dev
```
